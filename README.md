# DriveFleet — Online Vehicle Rental System

## Overview

DriveFleet is a static front-end web application for an online vehicle rental system. It provides a polished single-page interface to browse a vehicle fleet, book a rental, and interact with admin features.

## Features

- Responsive landing page with promotional hero section
- Vehicle catalog with filtering by category
- Multi-step booking flow for selecting a vehicle, entering customer details, and choosing rental dates
- Local storage data persistence for vehicles, customers, and rentals
- Admin dashboard support for managing bookings and vehicle inventory
- Toast notification UI for booking updates and form validation

## Project Structure

- `index.html` — Main application markup and page sections
- `style.css` — Visual styling and responsive layout rules
- `script.js` — Application logic, routing, data handling, and booking flow
- `assets/` — Vehicle images and static assets used by the site

## Getting Started

### Requirements

- Modern web browser (Chrome, Firefox, Edge, Safari)
- No backend required

### Run Locally

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. The app will load and initialize sample vehicle data automatically.

## How to Use

- Click the navigation links to move between Home, Vehicles, Rent, Return, and Admin sections.
- Use the `Rent Now` section to select a vehicle, enter customer details, and confirm rental dates.
- The app stores bookings in browser local storage, so refreshed visits retain data within the same browser.
